# Fantasy Log
Self-hosted rust app for world-building. 

## Local Development

1. Install the front-end.

```sh
pnpm install
```

2. Install the back-end.

```sh
cargo install --path src-tauri
```

3. Serve the app.

```sh
pnpm app-dev
```
